# Astoral AR Viewer

Bu klasör GitHub Pages üzerinde GLB/AR görüntüleme için hazırlandı.

## Kullanım

1. GLB dosyanızı bu klasöre koyun.
2. Dosya adını `model.glb` yapın.
3. iPhone AR için varsa USDZ dosyanızı da koyun ve adını `model.usdz` yapın.
4. GitHub reposunda dosyaları yükleyin.
5. Settings > Pages > Deploy from a branch > main / root seçin.
6. Yayın linkini telefonda açın.

## Dosyalar

- `index.html`: AR viewer sayfası
- `model.glb`: Sizin 3D modeliniz olacak
- `model.usdz`: iPhone için opsiyonel dosya
